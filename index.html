Pico

Construction de systèmes pour RPGMakerMV

Github Documentation

Pico est un petit plugin pour faciliter la construction de systèmes complexes en évènement avec le logiciel RPGMaker MV. Il est distribué sous licence MIT (X11).

Cette page fera office de page officielle pour le plugin. N'hésitez pas demander du support sur le forum RPGMaker Alliance ou via Discord.
